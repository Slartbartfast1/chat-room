基于nodejs,express,mongoDB,mongoose,socket.io,bootstrap的即时聊天室 总的来说有几个点: 

1:登陆注册功能,基于session的状态保持 

2.多房间聊天室,基于socket的命名空间和rooms功能. 

3.注册功能异步ajax表单提交. 

4.mongodb数据库的查询. 

5.表单提取中间件的使用. 

6.art-template模板引擎渲染页面. 

7.显示房间内的所有人,离开进入房间会发出提示
